Markdown的相对目录引用

发表于 2022-06-09 分类于 Markdown

Markdown的绝对目录引用可以去图床，各类图床可以在知乎上找到。分为付费和免费两种。
Markdown的相对目录是从你引用图片的目录开始，按级别搜索，往上一级搜索是./，往下一级搜索是直接写/filename(你的文件夹名)。
